# Revix

This repository is for learning and practicing Git and GitHub basics.

## Purpose
- Learn how to use Git
- Create repositories
- Push code to GitHub
- Track file changes

## Project Structure
src/main/java/   -> Java files
README.md       -> This file

## Getting Started

1. Clone the repo
- git clone <repo-link>

2. Make changes and push it
- git add .
- git commit -m "message"
- git push
